# 1. arr.push()

从后面添加元素，添加一个或多个，返回值为添加完后的数组长度

```javascript
let arr = [1, 2, 3, 4, 5]

console.log(arr.push(6, 7)) // 7
console.log(arr) // [1,2,3,4,5,6,7]
```
